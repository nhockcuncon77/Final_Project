## Troubleshooting

If images are not showing:

1. **Hard refresh your browser**: Press `Ctrl+F5` (Windows) or `Cmd+Shift+R` (Mac) to clear cache
2. **Restart the server**: Stop and restart `npm run serve`
3. **Check browser console**: Open Developer Tools (F12) and check for any 404 errors
4. **Verify file paths**: Make sure all image files exist in the `data/` folder with exact names (case-sensitive)
